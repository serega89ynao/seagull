== Description ==
Our 2018 default theme is clean, blog-focused, and designed for clarity. Seagull simple, straightforward typography is readable on a wide variety of screen sizes, and suitable for multiple languages. We designed it using a mobile-first approach, meaning your content takes center-stage, regardless of whether your visitors arrive by smartphone, tablet, laptop, or desktop computer.

* Mobile-first, Responsive Layout
* Custom Header
* Social Links
* Menu Description
* Post Formats

== Installation ==
1. In the admin panel go to Appearance> Themes and click the Add New button.
2. Click Download Theme and select File, then select Theme ZIP File. Click Install Now.
3. Click Activate to use the new theme immediately.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
